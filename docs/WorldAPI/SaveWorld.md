# Save World

This API allows you to save a world.

!> This API only works if you own the world.

## Request Method 
Create World
    POST

Update World
    PUT

## Endpoint
Create World
    https://api.vrchat.cloud/api/1/worlds

Update World
    https://api.vrchat.cloud/api/1/worlds/:id

id - the world's id

## Requires Authentication
Yes

## Parameters
The new World data

## Returns 
The updated world data
